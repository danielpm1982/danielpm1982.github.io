Apache Maven: How to Install and Test It on Linux

How to install and test Apache Maven on linux by creating standardized projects based on Maven archetypes; how to manage project lifecycle phases through plugins goals; how to manage external dependencies and even create and install your own dependencies on the local .m2 Maven repository. Also how to package and run a Java .jre file built with Maven. All this from the terminal. Includes a thorough view of Maven documentation at the end.
